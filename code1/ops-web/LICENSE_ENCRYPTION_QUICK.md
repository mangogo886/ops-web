# 许可加密功能快速说明

## ✅ 已实现功能

许可文件现已采用 **AES-256-CBC 加密**，防止手动修改。

## 🔒 加密特性

- ✅ 许可文件内容已加密，无法直接查看或修改
- ✅ 必须使用 `license-generator.exe` 工具生成许可文件
- ✅ 系统自动解密和验证，用户无需操作

## 📝 使用方法

### 生成加密许可文件

```bash
license-generator.exe -expire 2026-12-30 -mac 00:11:22:33:44:55 -output config/license.json
```

生成的 `config/license.json` 文件是加密的，无法手动编辑。

### 系统使用

系统会自动：
1. 读取加密的许可文件
2. 解密并验证许可
3. 检查授权日期和MAC地址

**用户无需任何操作**，系统会自动处理。

## ⚠️ 重要提示

1. **不能手动编辑许可文件**
   - 文件已加密，手动编辑会导致文件损坏
   - 系统无法解密损坏的文件，会拒绝登录

2. **只能使用工具生成**
   - 必须使用 `license-generator.exe` 生成
   - 不能手动创建或编辑 `config/license.json`

3. **文件损坏处理**
   - 如果文件损坏，删除后重新使用工具生成

## 🔧 技术实现

- **加密算法**：AES-256-CBC
- **密钥**：硬编码在代码中
- **编码**：Base64
